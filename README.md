### 1. Create the GitHub Repository
- Go to GitHub and create a new repository named exactly: `[your-username].github.io`
- e.g., if your username is `alipilehvar`, the repo must be `alipilehvar.github.io`

### 3. Enable GitHub Pages
- Go to repository **Settings → Pages**
- Under "Source", select **Deploy from a branch**
- Select branch: `main` (or `master`), folder: `/ (root)`
- Click **Save**
- Your site will be live at `https://[my-username].github.io` within ~2 minutes
